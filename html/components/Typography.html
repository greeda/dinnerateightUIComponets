Typography

폰트 정의
  1. 기본 폰트 사이즈는 10px으로 설정되어 있다. 보통의 경우 font-family 속성을 쓰지 않는다.
  2. 타이틀 태그 안에서 다른 종류의 폰트가 추가 될 경우 자식요소의 태그를 추가한다.
  3. 자식요소 태그에 스타일링은 상속받아 사용하지 않고 클래스를 추가한다.
  4. 볼드체와 이텔릭체는 클래스를 추가하여 사용한다. ex) .bold .italic
  5. 별도의 line-height, letter-spacing을 설정하지 않고 사용한다.
